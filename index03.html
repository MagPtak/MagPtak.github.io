Application's purpose is to allow user to insert expenses and save them in database. There are four types of expenses: grocery, household, work, and other. The app sums up each category, and all expenses together. Two different charts illustrate expenses in monthly and annually comparisons. Every expense has an edit and a remove option. There is also a login and a password validation and password reset option if a user forgot it. Application was teamwork. First of all, the team has splitted a project into twenty small tasks and group them into two sprints. For good work management Jira was used by a team. Data inserted by users is saved in Firebase database. My job was to: make routing, login view, login handling, registration handling, edit expenses view, edit expenses handling, remove expenses view, remove expenses handling.
Technologies used: React, JavaScript, Firebase, Styled-components, MUI.
